# Tips
1. 重现文献永远绕不开两个基本问题：你的模型和文献的是否相同或者足够相似？你的计算参数（DFT：基组、XC泛函、赝势、截断能、k点等；分子力学和MD：力场、电荷、MD参数：系综、恒温器、恒压器、时间步长等）是否和文献的完全相同？
2. 再精心的参数设置，再强大的计算机性能，再精致的结果输出和展示，如果构建的模型一开始是错误的，那么一切都是徒劳！
3. 代表性--抓住研究对象的关键特点，简化而不失真
4. 可用性--尺寸适中、计算量可接受，算得动
